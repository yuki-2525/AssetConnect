BOOTHダウンロードを自動追跡するブラウザ拡張機能

BOOTHからのダウンロードを自動で記録し、アセット管理ツールとの連携を簡単にするブラウザ拡張機能です。

※ AssetConnect のメインインターフェース

主な機能

📝 自動ダウンロード追跡

BOOTHのライブラリ、商品ページ、注文履歴、ギフトページでのダウンロードを自動で記録します。

📊 履歴管理

ダウンロードした商品名、ファイル名、日時、BOOTHIDを一覧で管理できます。

🎯 フィルタリング

無料ダウンロードのみ表示、同一IDのアイテムをまとめるなど、柔軟なフィルタリングが可能。

📤 エクスポート機能

CSV形式やAvatar Explorer Tools形式のJSONで履歴をエクスポートできます。
