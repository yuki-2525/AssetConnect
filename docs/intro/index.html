動作要件

必須環境

  • ブラウザ: Google Chrome 88以上 / Microsoft Edge 88以上
  • プラットフォーム: Windows 10/11, macOS 10.15+, Linux
  • アクセス権限: booth.pmドメインへのアクセス

連携対応ツール

  • Avatar Explorer: エクスポート機能で連携可能
  • KonoAsset: CSVエクスポートで連携可能

インストール手順

方法1: Chrome Web Storeからインストール (未実装)

1

Chrome Web Storeへアクセス

将来的にChrome Web Storeで公開予定です。現在は手動インストールをご利用ください。

💡 画像: Chrome Web StoreでのAssetConnectページ

方法2: 手動インストール (推奨)

1

拡張機能をダウンロード

GitHub Releasesから最新版の.zipファイルをダウンロードし、解凍します。

💡 画像: GitHub Releasesページでのダウンロード

2

デベロッパーモードを有効化

Chromeで chrome://extensions/ にアクセスし、右上の「デベロッパーモード」を有効化します。

💡 画像: Chrome拡張機能ページでデベロッパーモード有効化

3

拡張機能を読み込み

「パッケージ化されていない拡張機能を読み込む」をクリックし、解凍したフォルダを選択します。

💡 画像: 拡張機能の読み込み手順

4

インストール完了

AssetConnectが拡張機能一覧に表示されたらインストール完了です。

💡 画像: インストール完了状態の拡張機能一覧

拡張機能の設定

基本設定

1

拡張機能を起動

ブラウザのツールバーに表示されたAssetConnectアイコンをクリックします。

💡 画像: ブラウザツールバーのAssetConnectアイコン

2

ダウンロードフォルダを設定

ポップアップ画面でダウンロードフォルダのパスを入力し、「保存」ボタンをクリックします。

例: C:\Users\YourName\Downloads\BOOTH

💡 画像: ダウンロードフォルダ設定画面

フィルタリング設定

1

フィルタリングオプション

必要に応じて以下のオプションを設定します。

  • 無料ダウンロードのみ: 無料アイテムのみ表示
  • 同一IDのアイテムをまとめる: 同じ商品のファイルをグループ化
  • 一括登録: 同じ商品の全ファイルを一括追加

初回使用手順

1

BOOTHページにアクセス

BOOTHのライブラリ、商品ページ、注文履歴、ギフトページのいずれかにアクセスします。

💡 画像: BOOTHのライブラリページ

2

ダウンロードを実行

ダウンロードリンクをクリックすると、自動でダウンロード履歴が記録されます。

💡 画像: ダウンロードリンクをクリックする様子

3

履歴を確認

拡張機能のアイコンをクリックしてポップアップを開き、ダウンロード履歴が記録されていることを確認します。

💡 画像: ダウンロード履歴が表示されたポップアップ

4

エクスポートテスト

「CSV出力」または「JSON出力」ボタンをクリックしてエクスポート機能をテストします。

💡 画像: エクスポートボタンとダウンロードされたファイル

トラブルシューティング

⚠️ ダウンロードが記録されない

  • 拡張機能が正しくインストールされているか確認
  • BOOTHのダウンロードリンクを直接クリックしているか確認
  • ブラウザを再起動して再試行

🐛 拡張機能が表示されない

  • デベロッパーモードが有効化されているか確認
  • 拡張機能フォルダが正しく読み込まれているか確認
  • manifest.jsonにエラーがないか確認

💾 エクスポートできない

  • ダウンロードフォルダが正しく設定されているか確認
  • ブラウザのダウンロード許可が有効か確認
  • ポップアップブロッカーが無効化されているか確認

🔒 アクセス権限エラー

  • 拡張機能のアクセス権限がBOOTHドメインに許可されているか確認
  • Chromeのセキュリティ設定を確認
  • 拡張機能を無効化・有効化して再試行
