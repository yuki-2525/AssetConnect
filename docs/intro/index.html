インストール手順

方法1: Chrome Web Storeからインストール

1

Chrome Web Storeへアクセス

以下のリンクからChrome Web StoreのAssetConnectページにアクセスします。

AssetConnect - Chrome Web Store

💡 画像: Chrome Web StoreでのAssetConnectページ

2

拡張機能をインストール

「Chromeに追加」ボタンをクリックして拡張機能をインストールします。

💡 画像: 「Chromeに追加」ボタンをクリック

3

インストール確認

確認ダイアログで「拡張機能を追加」をクリックしてインストールを完了します。

💡 画像: インストール確認ダイアログ

4

インストール完了

AssetConnectアイコンがブラウザのツールバーに表示されたらインストール完了です。

💡 画像: ツールバーに表示されたAssetConnectアイコン

方法2: 手動インストール (非推奨)

1

拡張機能をダウンロード

GitHub Releasesから最新版の.zipファイルをダウンロードし、解凍します。

💡 画像: GitHub Releasesページでのダウンロード

2

デベロッパーモードを有効化

Chromeで chrome://extensions/ にアクセスし、右上の「デベロッパーモード」を有効化します。

💡 画像: Chrome拡張機能ページでデベロッパーモード有効化

3

拡張機能を読み込み

「パッケージ化されていない拡張機能を読み込む」をクリックし、解凍したフォルダを選択します。

💡 画像: 拡張機能の読み込み手順

4

インストール完了

AssetConnectが拡張機能一覧に表示されたらインストール完了です。

💡 画像: インストール完了状態の拡張機能一覧

拡張機能の設定

基本設定

1

拡張機能を起動

ブラウザのツールバーに表示されたAssetConnectアイコンをクリックします。

💡 画像: ブラウザツールバーのAssetConnectアイコン

2

ダウンロードフォルダを設定

ポップアップ上部の「ダウンロードフォルダのパス」欄にフォルダパスを入力し、「保存」ボタンをクリックします。

例: D:\users\Downloads

💡 画像: ダウンロードフォルダパス設定とグレーの設定エリア

フィルタリング・表示設定

1

表示オプションの設定

設定エリアのチェックボックスで表示方法をカスタマイズできます：

  • 無料ダウンロードのみ: 無料アイテムのみを表示してリストを絞り込み
  • 同一IDのアイテムをまとめる: 同じBOOTH IDの商品ファイルをグループ化して表示
  • 同一IDのアイテムを一括登録する: グループ化時に複数ファイルをまとめて外部ツール連携（上記オプション有効時のみ表示）

💡 画像: チェックボックス付きの白いトグル項目

初回使用手順

1

BOOTHページにアクセス

BOOTHのライブラリ、商品ページ、注文履歴、ギフトページのいずれかにアクセスします。

💡 画像: BOOTHのライブラリページ

2

ダウンロードを実行

ダウンロードリンクをクリックすると、自動でダウンロード履歴が記録されます。

💡 画像: ダウンロードリンクをクリックする様子

3

履歴を確認

拡張機能のアイコンをクリックしてポップアップを開き、ダウンロード履歴が記録されていることを確認します。

💡 画像: ダウンロード履歴が表示されたポップアップ

4

データ管理機能の確認

ポップアップ上部のボタンで以下の機能が利用できます：

  • JSON出力(AE Tools形式): AvatarExplorer連携用の形式でエクスポート
  • CSV出力: 表計算ソフトで扱いやすい形式でエクスポート
  • CSVインポート: 既存のCSVデータをインポート
  • 履歴削除: 記録されたダウンロード履歴をリセット

💡 画像: データ管理ボタン群（JSON出力・CSV出力・CSVインポート・履歴削除）

外部ツール連携

AvatarExplorer・KonoAsset連携

1

連携ボタンの確認

各ダウンロード項目の右側に、以下の連携ボタンが動的に表示されます：

  • AvatarExplorer ボタン: vrcae://プロトコルでAvatarExplorerツールに連携
  • KonoAsset ボタン: konoasset://プロトコルでKonoAssetツールに連携

ボタンの色:

  • 青色：通常状態
  • オレンジ色：未登録状態（外部ツールに登録されていない項目）

💡 画像: 各項目右側の青・オレンジの連携ボタン表示

2

連携機能の活用

これらのボタンをクリックすることで、アバターアセット管理ツールと連携し、より効率的なアセット管理が可能になります。

ヒント: JSON出力(AE Tools形式)と合わせて使用することで、より詳細なアセット情報を活用できます。

💡 画像: 連携ツール使用例

その他の機能

1

言語設定

ポップアップ下部の「Language」セレクタで表示言語を変更できます：

  • 日本語
  • English
  • 한국어

💡 画像: 下部固定バーの言語セレクタ

2

追加機能

ポップアップ下部のリンクから以下の機能にアクセスできます：

  • タブで表示: ポップアップを新しいタブで開いてより広い画面で操作
  • 更新履歴: 拡張機能のアップデート内容を確認
  • 作者X: 開発者のXアカウントへのリンク
  • お問い合わせ: フィードバックフォームへのリンク

💡 画像: 下部固定バーの各種リンク

トラブルシューティング

⚠️ ダウンロードが記録されない

  • 拡張機能が正しくインストールされているか確認
  • BOOTHのダウンロードリンクを直接クリックしているか確認
  • ブラウザを再起動して再試行

🐛 拡張機能が表示されない

  • デベロッパーモードが有効化されているか確認
  • 拡張機能フォルダが正しく読み込まれているか確認
  • manifest.jsonにエラーがないか確認

💾 エクスポートできない

  • ダウンロードフォルダが正しく設定されているか確認
  • ブラウザのダウンロード許可が有効か確認
  • ポップアップブロッカーが無効化されているか確認

🔒 アクセス権限エラー

  • 拡張機能のアクセス権限がBOOTHドメインに許可されているか確認
  • Chromeのセキュリティ設定を確認
  • 拡張機能を無効化・有効化して再試行
